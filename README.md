# 📊 SALES PERFORMANCE DASHBOARD

## Project Overview

This project demonstrates an interactive Sales Performance Dashboard built using Microsoft Power BI.

## Tools Used

- Power BI Desktop
- Microsoft Excel
- DAX
- Power Query

## Dashboard Preview

![Dashboard](Images/Dashboard.png)

## KPIs

- Total Sales
- Total Profit
- Total Orders
- Average Sales

## Dashboard Features

- Monthly Sales Trend
- Sales by Region
- Profit by Category
- Top 10 Products
- Sales by State
- Customer Segment
- Interactive Slicers

## Dataset

- 200 Sales Records
- Excel Dataset

## Skills Demonstrated

- Data Cleaning
- Power Query
- Data Modeling
- DAX Measures
- Data Visualization
- Dashboard Design
