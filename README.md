# 📊 SALES PERFORMANCE DASHBOARD

## 📌 PROJECT OVERVIEW

The Sales Performance Dashboard is an interactive Power BI project developed to analyze sales data and provide meaningful business insights. It enables users to monitor key performance indicators (KPIs), identify sales trends, evaluate regional performance, and make data-driven decisions.

---

## 🎯 OBJECTIVES

- Analyze overall sales performance.
- Monitor profit and order trends.
- Compare sales across regions and states.
- Identify top-performing products and categories.
- Provide an interactive dashboard for business users.

---

## 🛠️ TOOLS & TECHNOLOGIES

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel

## 📌 KEY INSIGHTS

- Total Sales exceeded ₹416K.
- Total Profit reached ₹56K.
- Total Orders recorded: 167.
- Corporate segment generated the highest sales.
- South region achieved the highest sales performance.
- Phones, Tables, and Binders were among the top-selling products.
- Office Supplies contributed the highest profit.
